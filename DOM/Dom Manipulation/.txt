DOM Manipulation :
- selecting with id : document.getElementById("myId")   // getElementById() is a method in document object 
- selecting with class : document.getElementsByClassName("myClass") 
- selecting with tag : document.getElementByTagName("p")

-- we can level-up for selecting elements without using above document methods()
- we use Query Selector
  -  document.querySelector("myId / myClass / tag")  // returns first element
  -  document.querySelectorAll("myId / myClass / tag")  // returns a NodeList


DOM Manipulation : properties (we can get, change/update the values using properties)
  - tagName : returns tag for element NodeList
  - innerText : returns the text content of the element and all its children
  - innerHtml : returns the plain text or HTML contents in the element
  - textContent : returns textual content even for hidden elements
